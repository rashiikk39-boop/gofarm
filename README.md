# 🌾 GoFarm — Modern Agricultural E-Commerce Platform

A production-ready agricultural marketplace built with Next.js, React, TypeScript, and Tailwind CSS.

---

## 🚀 Overview

GoFarm is a modern full-stack agricultural e-commerce platform designed for selling fresh farm products online. The application includes product management, authentication, payment integration, inventory tracking, and a fully responsive shopping experience.

Built with scalability and performance in mind, the platform follows modern development practices using the latest Next.js App Router architecture.

## 📦 Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/gofarm.git
cd gofarm
```

### Install Dependencies

```bash
pnpm install
```

### Configure Environment Variables

```bash
cp .env.example .env.local
```

Add your Firebase, Stripe, and CMS credentials inside `.env.local`.

### Start Development Server

```bash
pnpm dev
```

Visit:

```bash
http://localhost:3000
```
